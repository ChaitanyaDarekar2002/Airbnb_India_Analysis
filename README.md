# 🏡 Airbnb India Data Analysis Project

## 📌 Overview
This project presents an end-to-end data analysis solution for Airbnb listings in India using **Python, SQL Server, and Power BI**. The analysis focuses on pricing trends, room types, ratings, guest capacity, and superhost performance to generate meaningful business insights and interactive dashboards.

---

# 🎯 Project Objectives

✅ Perform data cleaning and preprocessing  
✅ Analyze Airbnb pricing trends  
✅ Compare room types and guest capacity  
✅ Identify top-rated and luxury properties  
✅ Analyze superhost performance  
✅ Build interactive Power BI dashboards  
✅ Create a portfolio-ready analytics project
